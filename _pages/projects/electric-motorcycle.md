# Electric Motorcycle Control System

<img src="/images/ghost_controls.jpg" alt="GHOST Electric Motorcycle" style="width: 100%; max-width: 800px; border-radius: 8px; margin-bottom: 20px;">

## Overview

This project focused on developing a comprehensive control system for a high-performance electric motorcycle powered by a 45kW PMAC (Permanent Magnet AC) motor operating at 103.6V. The system integrates multiple sensors and implements sophisticated feedback control for optimal performance.

## Technical Details

**Skills Used:** Motor Control, Sensor Integration, CAN Bus, Embedded Systems, Real-time Control

### System Specifications

- **Motor**: 45kW PMAC motor with field-oriented control
- **Voltage**: 103.6V battery system
- **Control Strategy**: Torque-based feedback control with real-time optimization
- **Communication**: CAN bus protocol for sensor integration and data logging

### Key Achievements

- **Motor Control Software**: Developed control algorithms optimizing torque response and efficiency
- **Sensor Integration**:
  - Motor encoder for precise speed and position feedback
  - IMU for vehicle dynamics monitoring
  - Temperature sensors for thermal management
  - BMS integration for battery state monitoring
- **Performance Dashboard**: Real-time visualization of vehicle parameters and performance metrics
- **Electrical System**: Custom-designed and fabricated wiring harness and high-power cabling meeting safety standards

## Control Architecture

The control system implements:
- Feedback control loops for torque and speed regulation
- Throttle mapping for smooth power delivery
- Regenerative braking control
- Thermal management for motor and battery protection
- Safety interlocks and fault detection

## Technical Challenges

- Managing high current switching in power electronics
- Implementing robust noise filtering for sensor signals in high-EMI environment
- Optimizing control parameters for performance and efficiency
- Designing reliable high-power electrical connections

## Results

The control system successfully enables smooth, responsive control of the electric powertrain with real-time monitoring of all critical vehicle systems. The integrated dashboard provides riders with comprehensive performance data while maintaining safe operating conditions.
