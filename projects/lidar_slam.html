Project Overview

Developed a comprehensive mobile manipulator simulation in ROS 2 Humble, integrating hardware from multiple manufacturers into a cohesive robotic system. This project demonstrates advanced robotics concepts including LIDAR-based SLAM (Simultaneous Localization and Mapping), autonomous navigation, and coordinated mobile manipulation. The system combines a Clearpath Husky mobile base, Universal Robots UR3 arm, and Robotis gripper into a fully functional autonomous platform.

Integration Architecture

The mobile manipulator represents a complex integration challenge, requiring coordination between the mobile base navigation system and the arm manipulation controllers. Custom URDF models were created to accurately represent the combined system's kinematics, including proper mass properties, inertia tensors, and collision geometries.

Technical Challenges & Solutions

Challenge: Multi-Platform Integration

Problem: Integrating hardware from different manufacturers with varying URDF formats and coordinate frame conventions.
Solution: Created unified URDF with proper coordinate transformations, validated using RViz TF visualization and Gazebo physics simulation.

Challenge: Controller Coordination

Problem: Managing multiple controllers (base, arm, gripper) without conflicts.
Solution: Implemented ros2_control framework with proper controller namespacing and resource management.

Challenge: SLAM Parameter Tuning

Problem: Achieving reliable SLAM performance in simulation environment.
Solution: Systematically tuned SLAM Toolbox parameters including resolution, correlation search space, and loop closure detection thresholds.
